🛠 SQL para activar hotel_id en nh_config

Si el test "Columna hotel_id" falla, ejecuta este SQL en el SQL Editor de Supabase. Solo necesitas hacerlo una vez.

-- 1. Añadir columna hotel_id a nh_config ALTER TABLE nh_config ADD COLUMN IF NOT EXISTS hotel_id int8 DEFAULT 1; -- 2. Asignar hotel 1 a los datos existentes UPDATE nh_config SET hotel_id = 1 WHERE hotel_id IS NULL; -- 3. Índice para mejorar rendimiento CREATE INDEX IF NOT EXISTS idx_nh_config_hotel_id ON nh_config(hotel_id); -- 4. (Opcional) Limpiar filas duplicadas si las hay DELETE FROM nh_config WHERE id NOT IN ( SELECT MAX(id) FROM nh_config GROUP BY key, hotel_id );
